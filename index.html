How It Works

In this matrix-like domain, each Cat is produced by hashing your wallet address + an optional ID, then mapping that hash to ASCII cat attributes.

  1. Connect Wallet: Use your Phantom wallet to link into the system.
  2. ID Input: Provide a unique ID or default to your address alone.
  3. Hashing Protocol: Transforms your input into a 32-bit integer we use for cat data.
  4. Cat Attributes: We assign breed, fur color, trait, accessory, and “purr level.”
  5. Deterministic Output: The same inputs yield the same ASCII cat, exclusively yours!

This solution merges cryptographic determinism with retro terminal flair.
